# VivaMente 

## Descrição do projeto
Projeto Full Stack focado em saúde mental chamado **VivaMente**, desenvolvido com Node.js, MySQL no backend, com frontend em HTML, CSS e JavaScript, e preparado para futura versão em Flutter.

O sistema permite o gerenciamento de conteúdos de bem-estar como meditação, respiração, hábitos saudáveis e autocuidado.

---

## Tecnologias

### Backend:
- Node.js
- MySQL

### Frontend:
- HTML
- CSS
- JavaScript

### Ferramentas:
- VS Code
- Insomnia
- MySQL

---

## Funcionalidades
- Listar conteúdos de saúde mental
- Criar novos conteúdos
- Editar conteúdos existentes
- Remover conteúdos
- Visualizar detalhes de cada conteúdo
- Estrutura pronta para usuários, favoritos e comentários
